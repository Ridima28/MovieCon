# 🎬 React Movie Search App

A dynamic movie search application built using React that allows users to search for movies and explore details in real time using an external movie API.

---

## 🚀 Overview

This project was built by following a React tutorial and then extending it with custom improvements to better understand:

* React components & props
* State management using hooks
* API integration
* Conditional rendering

---

## ✨ Features

* 🔍 Search movies by title
* 🎞️ Display movie posters and details
* ⚡ Real-time API fetching
* 📦 Component-based architecture
* 📱 Responsive UI

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS
* Movie API (TMDB)
